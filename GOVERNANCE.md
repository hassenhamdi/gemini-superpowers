# Project Governance

Gemini Superpowers is a community-driven project that relies on the participation and contributions of its members.

## Governance Model

The project follows a **benevolent dictator for life (BDFL)** model, where Hassen Hamdi has final decision-making authority over the project's direction.

## Core Maintainers

Hassen Hamdi is the current core maintainer of the project.

## Decision Making

All significant technical decisions will be made through open discussion on GitHub issues and pull requests.
Hassen Hamdi will make the final decision if no consensus can be reached.

## New Maintainers

We welcome new maintainers who have demonstrated a commitment to the project and its core philosophy.
New maintainers will be invited by Hassen Hamdi.

## Code of Conduct

All contributors and maintainers are expected to adhere to the [Code of Conduct](CODE_OF_CONDUCT.md).
